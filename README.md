# Expressly Notes  

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Description

Expressly Notes is an application for note taking.  The application allows the user to create notes and then read them by selecting them from the list of all notes displayed in the left-most column.  Notes can also be deleted from within this application.  This application is a deployed application via the Heroku service.  The front-end of this application was designed by https://github.com/coding-boot-camp/miniature-eureka  and the code from this repository was used to begin the application development.  The backend of the application is written using Express.js and a json file to store the data displayed in the application.

Link to deployed application: [https://moebirdie.github.io/Inquirer-README-Generator/](https://moebirdie.github.io/Inquirer-README-Generator/)

## Features <a id="Features"></a>

Features exhibited in this application include simple question prompts, editor prompts to enable paragraph style input and automated README generation..
  

## Usage <a id="Usage"></a>

This application is intended to be used by an end user to create a professional README file.  By prompting the user to answer questions, the end user is ensured to produce all the necessary elements of a good README file.  The user is expected to answer questions, then the questions are used by the create README file and a README is created in the repository.
